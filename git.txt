git commands
1> to view commited files and current location of head->
git log --oneline 
* oneline is not necessary but fetches us only required info

2> to move to some previous comitted file i.e move head position
git reset --hard HEAD~1
* 1 moves it one position back and hard makes this change everywhere
3> to check  status of all current files
git status -s 
here also s is optional but fetches us only required info 
if a file is commited then git status treats it as a pure file and does not return anything about it i.e no status for that purpose we use git log command
4> to add a branch 
git branch branchname
5> to view current branches 
git branch
6>to enter into a branch
git switch branch name
we can also use it to switch to main
7>to merge branch to main code go to main code and
git merge branchname
8>to know the path followed while merging
git log --oneline --graph
9>to delete a branch
git branch -d branchname
